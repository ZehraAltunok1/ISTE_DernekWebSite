# İSTE UI5 Dernek Platformu

Bu proje, SAPUI5 kullanılarak geliştirilen bir dernek yönetim platformudur.

## Amaç

Üniversite öğrencileri için burs ve bağış süreçlerini yönetebilen,
rol bazlı (public / auth / admin) bir web uygulaması geliştirmek.

## Kullanılan Teknolojiler

- SAPUI5 (OpenUI5)
- Node.js (LTS)
- UI5 CLI
- JavaScript
- XML (UI5 View)
- JSON
